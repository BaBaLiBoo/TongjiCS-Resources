# 同济大学计算机专业学习资源库

## 📚 仓库简介

本仓库是同济大学计算机科学与技术专业本科生的学习资源集合，旨在整理和分享课程相关的学习资料，包括电子版教材、习题解析、历年真题试卷、整理的题库以及复习资料等。

## 📖 资源说明

本仓库按照课程分类组织，课程文件夹包含以下类型的资源：

- 📘 **电子版教材**：课程使用的教材电子版
- 📝 **习题解析**：课后习题的详细解答
- 📄 **历年真题试卷**：往年的期末考试试卷及参考答案
- 📋 **整理的题库**：根据课程内容整理的题目集合
- 📚 **复习资料**：课程笔记、复习提纲等学习辅助材料
